# Personal Portfolio Website

##  Overview

This is my personal portfolio website developed to showcase my technical skills, academic projects, certifications, and contact information. The website features a modern responsive design that works across desktop and mobile devices.

##  Features

- Responsive Design
- Professional Home Section
- About Me
- Skills Section
- Projects Showcase
- Certifications
- Contact Information
- GitHub & LinkedIn Links
- Smooth Navigation

##  Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

##  How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using a web browser.
